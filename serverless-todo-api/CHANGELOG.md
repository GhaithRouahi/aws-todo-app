# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-22

### Added
- Initial release of the Serverless To-Do API.
- Core CRUD functionality for to-do items (Create, Read, Update, Delete).
- AWS SAM template for infrastructure as code.
- GitHub Actions workflow for CI/CD.
- Basic frontend application.
- OpenAPI specification.
- README.md with detailed project documentation.
